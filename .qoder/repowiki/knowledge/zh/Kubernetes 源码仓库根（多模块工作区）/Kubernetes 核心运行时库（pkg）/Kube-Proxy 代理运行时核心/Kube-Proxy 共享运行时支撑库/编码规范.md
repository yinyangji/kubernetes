- API 类型使用 `+k8s:deepcopy-gen=package` 与 `+groupName=` 注释声明，并由 zz_generated 文件自动生成 DeepCopy 与转换代码。
- 平台相关实现通过 `//go:build linux` 构建标签隔离，并在同目录下提供 `fake.go` 作为跨平台测试替身。
- 外部系统依赖（HTTP server、监听器、clock、node 访问）通过内部 interface（如 `listener`、`httpServerFactory`、`nodeManager`）注入，避免直接 import 顶层 `pkg/proxy` 造成循环依赖。
- Prometheus 指标统一以 `kubeproxy` 为 Subsystem，按 ProxyMode 在 `RegisterMetrics` 中 switch 分支注册，指标名遵循 `sync_proxy_rules_*` / `proxy_healthz_total` 等前缀约定。
- 配置项使用指针类型（如 `*int32`、`*bool`、`metav1.Duration`）区分未设置与零值，并通过 `defaults.go` 与 `zz_generated.defaults.go` 注入默认值。
