- 跨平台实现通过 Go build tag 拆分为 *_linux.go / *_windows.go / *_unsupported.go，共享同一 package 名与对外接口。
- 状态写入采用函数式 Setter 模式：导出构造函数返回 `func(ctx context.Context, node *v1.Node) error`，由调用方按需组合多个 Setter 完成一次 Node 更新。
- 指标统一通过 `k8s.io/component-base/metrics` 的 NewGauge/NewCounter/NewHistogram 系列在包级 var 中声明，并按 subsystem="kubelet" 命名。
- 测试辅助使用 mockery 生成 mock（`.mockery.yaml` + `//go:generate mockery`），并在 `testing/` 子目录或 `fake_*` 文件中提供 fake 实现。
- 对需要幂等写入的状态结构体（如 versionedPodStatus）使用版本号 + `isPodStatusByKubeletEqual` 比较来避免重复 patch API。
