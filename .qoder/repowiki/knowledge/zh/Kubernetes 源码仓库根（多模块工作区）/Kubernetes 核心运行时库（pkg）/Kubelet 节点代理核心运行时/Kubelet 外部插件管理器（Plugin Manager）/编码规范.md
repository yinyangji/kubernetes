- 每个对外能力都先定义 interface（如 PluginManager、Reconciler、OperationExecutor、PluginHandler），再以同名小写结构体实现并用 `var _ Interface = &impl{}` 做编译期校验。
- 构造统一通过 `NewXxx(...)` 工厂函数返回接口类型，内部以组合方式注入依赖，避免直接 new 结构体。
- 异步操作统一通过 goroutinemap 按 socketPath 键串行化，并对 IsAlreadyExists 与 IsExponentialBackoff 两类错误进行显式忽略，其余错误才记录日志。
- Register/Unregister 流程中先更新 actual state of world 再调用下游 handler，确保中间被删除事件能正确切换为 DeRegister 语义。
