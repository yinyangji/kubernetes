- 对外能力均以 `interface{}` + `New*` 构造器形式暴露，具体结构体以小写开头（如 `basicManager`、`manager`、`handlerRunner`），便于测试注入 mock。
- 并发安全采用 `sync.RWMutex` 保护内部 map 索引，读多写少场景下读写方法成对加锁（如 `workerLock`、`lock`）。
- 探针/生命周期钩子的失败路径统一记录结构化日志（使用 `klog.FromContext(ctx)` 获取 logger）并递增对应 metrics，同时可选地通过 `record.EventRecorder` 上报事件。
- 平台相关能力通过 build tag 拆分文件（`features_linux.go` / `features_windows.go` / `features_unsupported.go`），同一 package 内提供相同函数签名。
