- 针对平台差异使用 Go build 标签拆分实现：同名包内放置 `<feature>_linux.go`/`<feature>_unsupported.go` 或 `<feature>_unix.go`/`<feature>_windows.go`，公共接口放在无标签文件中。
- 对外暴露可注入的接口（如 `iptables.Interface`、`filesystem.Filesystem`、`procfs.ProcFSInterface`），并通过 `New*` 构造函数返回具体实现，测试侧用 `testing/fake.go` 或 `*_fake.go` 替换。
- 自定义 `pflag.Value` 类型统一实现 `Set`/`String`/`Type` 三个方法，并在文件顶部用 `var _ pflag.Value = &T{}; var _ pflag.Value = T{}` 编译期断言。
- 对 Kubernetes API 对象的修改函数一律返回新副本而非原地修改，并以 `(updated, error)` 或 `(newObj, changed, err)` 形式表达变更语义。
- 错误类型采用自定义结构体 + `Error() string` 实现 `error` 接口，并配套 `IsXxx(err) bool` 判断函数以便调用方分支处理。
