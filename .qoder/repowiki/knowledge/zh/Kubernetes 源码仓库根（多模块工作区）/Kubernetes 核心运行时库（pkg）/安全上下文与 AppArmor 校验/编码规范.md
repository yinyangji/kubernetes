- Accessor/Mutator 成对定义接口并以 `*Wrapper` 结构体实现，通过 `ensure*` 方法在写入时惰性初始化底层对象，避免 nil 指针解引用。
- 平台相关逻辑通过 `//go:build linux` / `!linux` 等 build tag 拆分为同名文件（如 `validate_disabled.go`、`util_linux.go`、`util_windows.go`），保持包 API 一致。
- 有效值合成遵循“容器字段优先于 Pod 字段”的覆盖规则，并在 `effectiveContainerSecurityContextWrapper` 中以 `reflect.DeepEqual` 比较后写回，避免无意义变更。
- 运行时默认路径集合使用 `sync.OnceValue` 或包级 `var` 延迟计算并缓存，防止重复 I/O。
