- 每个版本子包（如 `v1`）在 `doc.go` 中集中声明 `+k8s:deepcopy-gen`、`+k8s:conversion-gen`、`+k8s:defaulter-gen` 等代码生成注解，以驱动 zz_generated 文件产出。
- 内部类型通过 `register.go` 中的 `SchemeBuilder` 显式 `AddKnownTypes` 注册，并导出 `AddToScheme` 供上层全局 scheme 合并。
- 版本化配置通过 `latest.Default()` 先构造外部版本对象、调用 `scheme.Scheme.Default` 再 `Convert` 到内部类型，并在返回前回填 `TypeMeta.APIVersion`。
- 验证函数统一使用 `field.NewPath(...)` 构建路径并按 `utilerrors.Aggregate` 收集错误，而非直接返回单个 error。
- 已移除/废弃插件通过 `invalidPluginsByVersion` 列表按 `schemeGroupVersion` 分段声明，由 `isPluginInvalid` 根据当前 `APIVersion` 判定是否非法。
