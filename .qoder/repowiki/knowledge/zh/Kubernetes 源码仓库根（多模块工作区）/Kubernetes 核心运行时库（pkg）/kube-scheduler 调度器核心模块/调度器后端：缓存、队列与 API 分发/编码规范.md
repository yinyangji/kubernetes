- 对外构造统一使用 `NewXxx(..., opts ...Option)` + `With*` 函数注入 clock、listers、feature flags 等可测试依赖。
- 所有公开方法在需要日志时显式接收 `logger klog.Logger` 参数，而非全局获取，便于测试注入。
- 多锁结构采用严格的全局锁序注释（如 `lock > activeQueue.lock > backoffQueue.lock > nominator.nLock`），并在入口显式 `p.lock.Lock()/Unlock()`。
- 可选能力通过 `utilfeature.DefaultFeatureGate.Enabled(features.Xxx)` 布尔字段缓存到结构体中，避免每次路径重复查 feature gate。
- API 写操作统一走 `fwk.APIDispatcher`（BindPod、PatchPodStatus），仅在 feature gate 开启时赋值非 nil，保持同步路径不变。
