- 每个指标在声明时显式设置 `Subsystem: SchedulerSubsystem` 与 `StabilityLevel`（STABLE/BETA/ALPHA），并在 `InitMetrics` 中集中构造。
- 按 FeatureGate 条件注册的指标在 `Register()` 中以 `if utilfeature.DefaultFeatureGate.Enabled(features.Xxx)` 分支包裹，未启用则不加入 `metricsList`。
- 对同一业务结果的度量采用“计数器+直方图”成对更新模式（如 `scheduleAttempts` + `schedulingLatency`、`podGroupScheduleAttempts` + `podGroupSchedulingLatency`），并由专用 helper 函数统一封装。
- 面向调用方的写入 API 以 `With(metrics.Labels{...})` 或 `WithLabelValues(...)` 预绑定标签，再返回具体 metric 句柄供上层直接 Inc/Observe/Set。
- 异步记录器通过无阻塞 `select ... default` 向缓冲 channel 推送，满则丢弃，避免阻塞调度关键路径。
