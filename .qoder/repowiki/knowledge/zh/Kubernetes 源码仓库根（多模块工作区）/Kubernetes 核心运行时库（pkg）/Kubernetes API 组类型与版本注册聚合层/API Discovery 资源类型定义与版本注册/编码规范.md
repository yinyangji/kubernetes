- 每个版本子包通过 `localSchemeBuilder = &apidiscoveryv2(SchemeBuilder)` 复用外部包的 SchemeBuilder，再暴露本地 `AddToScheme`，避免重复实现注册逻辑。
- 根包与版本包均导出 `GroupName` 常量与 `SchemeGroupVersion` 变量，并配套提供 `Kind()` / `Resource()` 辅助函数构造 `GroupKind` / `GroupResource`。
- 所有对外可序列化的结构体都附带 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解，由 code-generator 统一生成 DeepCopy 实现。
