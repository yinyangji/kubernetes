- 每个公开版本子包（v1/v1beta1）都提供独立的 `register.go` 暴露 `GroupName`、`SchemeGroupVersion` 和 `AddToScheme`，并在 `init()` 中手动注册手写函数，使缺失生成文件时仍可编译。
- 根包 `types.go` 作为唯一类型来源，所有版本共享同一份结构体定义，版本差异仅体现在转换/默认值/验证的生成文件中。
- 校验函数统一以 `*field.Path` 作为字段路径前缀，通过 `field.NewPath(...).Child(...)` 逐级拼接，并以 `field.ErrorList` 累积错误返回。
- 可选项通过指针类型（如 `*int32`、`*bool`）表达，配合 `ptr.Deref` 与 `apiequality.Semantic.DeepEqual` 区分未设置与零值语义。
- 受支持的枚举值集中在包级 `sets.New(...)` 变量中（如 `supportedPodFailurePolicyActions`），校验时使用 `Has`/`NotSupported` 而非硬编码分支。
