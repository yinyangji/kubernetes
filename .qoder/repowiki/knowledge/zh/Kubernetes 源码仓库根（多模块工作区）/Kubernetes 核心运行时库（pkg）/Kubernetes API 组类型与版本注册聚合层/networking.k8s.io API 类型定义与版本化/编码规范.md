- 每个公开版本包（v1、v1beta1）都独立定义 `GroupName`、`SchemeGroupVersion`、`Resource()` 以及 `AddToScheme`，并在 `init()` 中仅注册手写函数，让生成文件可缺失编译。
- 对外资源类型统一嵌入 `metav1.TypeMeta` 与 `metav1.ObjectMeta`，并通过 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解标记 Object/List 配对。
- 校验函数成对出现 Create/Update 入口（如 `ValidateIngressCreate`/`ValidateIngressUpdate`），内部共用一个带 Options 结构体的私有 `validateXxx` 函数，通过旧对象推导放宽策略以实现向后兼容。
- 版本间不兼容字段通过 `vN/conversion.go` 中的 `Convert_vN_XXX_To_networking_XXX` 手动覆写 `autoConvert_*` 的方式处理，而非直接修改内部类型。
- 不可变字段更新校验统一使用 `apivalidation.ValidateImmutableField(...).MarkCoveredByDeclarative().WithOrigin("immutable")` 模式返回错误。
