- 每个外部版本子包都遵循固定文件集：`doc.go`（含 codegen 注释）、`register.go`（定义 GroupName/SchemeGroupVersion/AddToScheme）、`defaults.go`、`zz_generated.*` 三件套，便于代码生成与手动扩展分离。
- 对外暴露的 API 对象均附带 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注释，使其自动实现 runtime.Object 接口。
- 多态字段使用 `+union` 注解配合指针字段（如 Basic/Gang、Single/All）表示“恰好一个变体”的选择语义，并在注释中说明创建后不可变更。
- 受特性门控控制的字段通过 `+featureGate=<GateName>` 注解标注，使 code-generator 能据此生成条件编译或文档信息。
- 列表字段统一使用 `+listType=map` + `+listMapKey=name` 组合以支持 merge patch，必要时附加 `+patchMergeKey`/`+patchStrategy`。
- 系统保留常量（如 `SystemPriorityClassPrefix`、`HighestUserDefinablePriority`、`WorkloadMaxTemplateDepth`）集中在顶层 `types.go` 的 `const` 块中，由校验层引用而非硬编码。
