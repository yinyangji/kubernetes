- 每个 API 资源类型均标注 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object`，使其实现 Object 接口以便进入 Scheme。
- 内部版本放在根包并使用 `runtime.APIVersionInternal`，外部稳定版本拆入 `v1`/`v1beta1` 子包，各自独立定义 `SchemeGroupVersion` 与 `AddToScheme`。
- 对外版本的类型定义位于 `k8s.io/api/authorization/vX`，本模块的 `vX/register.go` 仅引用该包并重用其 `SchemeBuilder`，手写逻辑集中在 `defaults.go` 与 `register.go` 中。
- 校验函数成对出现：手写 `Validate*` 负责业务约束，再调用 `rest.DeclarativeValidation.ValidateDeclaratively` 叠加 `zz_generated.validations.go` 的规则，统一以 `Validate*Create(ctx, scheme, obj)` 暴露。
- Union 字段（如 ResourceAttributes/NonResourceAttributes、LabelSelectorAttributes.RawSelector/Requirements）采用“二者必居其一”的手写断言，并用 `WithOrigin("union")` 标记错误来源。
