本模块是 Kubernetes 内建 authorization.k8s.io API 组的纯类型定义层，按 apimachinery 约定分层：
- 根包 `pkg/apis/authorization`：声明内部版本（`runtime.APIVersionInternal`）的 Go 结构体（`SubjectAccessReview`、`SelfSubjectAccessReview`、`LocalSubjectAccessReview`、`SelfSubjectRulesReview` 及其 Spec/Status），并通过 `register.go` 中的 `SchemeGroupVersion`、`Kind`/`Resource` 辅助函数以及 `addKnownTypes` 向 Scheme 注册。
- `v1/`、`v1beta1/`：外部稳定版本，仅做 GroupVersion 常量、`AddToScheme` 包装和默认值注册；实际类型来自 `k8s.io/api/authorization/vX`，通过 `zz_generated.*` 文件生成 conversion、defaults、validations 代码，手写 `defaults.go`/`register.go` 仅补充非生成逻辑。
- `install/install.go`：聚合入口，在 `init()` 中把 internal、v1beta1、v1 全部注入到 `legacyscheme.Scheme`，并设置 v1 优先于 v1beta1。
- `validation/validation.go`：手写字段级校验（互斥 union、metadata 为空约束、namespace 一致性、RawSelector/Requirements 互斥等），再通过 `rest.DeclarativeValidation` 组合 `zz_generated.validations.go` 的声明式规则，统一暴露 `Validate*Create` 入口。
- `fuzzer/fuzzer.go`：为 fuzzing 提供随机对象构造。
依赖方向单向向下：v1/v1beta1 → 根包 internal types → apimachinery；install 反向聚合所有版本。
