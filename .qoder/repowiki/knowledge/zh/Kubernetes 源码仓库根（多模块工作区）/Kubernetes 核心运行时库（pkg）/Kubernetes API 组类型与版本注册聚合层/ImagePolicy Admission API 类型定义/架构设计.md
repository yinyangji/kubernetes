本模块是 imagepolicy.k8s.io API Group 的类型声明层，遵循 kubernetes 标准 apimachinery 约定：
- `types.go` 定义核心对象 `ImageReview` / `ImageReviewSpec` / `ImageReviewContainerSpec` / `ImageReviewStatus`，并通过 `+k8s:deepcopy-gen` 和 `runtime.Object` 标记生成 deepcopy；`register.go` 暴露内部组 `imagepolicy.k8s.io` 的 `SchemeGroupVersion`（使用 `runtime.APIVersionInternal`）以及 `Kind`/`Resource` 构造器。
- `v1alpha1/` 子包提供公开版本，其 `register.go` 通过 `localSchemeBuilder = &imagepolicyv1alpha1.SchemeBuilder` 复用 `k8s.io/api/imagepolicy/v1alpha1` 中生成的 SchemeBuilder，并手动注册 `RegisterDefaults`。
- `install/install.go` 作为入口，在 `init()` 中将 imagepolicy 内部组与 v1alpha1 注册到 `legacyscheme.Scheme`，并设置 `v1alpha1` 为最高优先级版本。
- `zz_generated.*` 文件由 `generate-groups.sh` 自动生成 deepcopy、conversion、defaults、validations；`fuzzer/fuzzer.go` 提供 fuzzing 支持。
依赖方向单向：install → v1alpha1 + imagepolicy 内部组 → apimachinery runtime/scheme，不反向引用上层业务逻辑。
