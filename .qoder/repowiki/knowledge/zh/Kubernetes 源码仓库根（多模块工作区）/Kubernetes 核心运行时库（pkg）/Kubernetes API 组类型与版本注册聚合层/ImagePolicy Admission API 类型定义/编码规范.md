- 每个版本子包通过 `var localSchemeBuilder = &<imported>.SchemeBuilder` 复用 apimachinery 生成的 SchemeBuilder，再暴露本地 `AddToScheme` 别名，使代码在缺少生成文件时仍可编译。
- API 对象使用 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注释驱动 deepcopy 生成，而非手写 DeepCopyInto。
- 内部组使用 `runtime.APIVersionInternal` 作为版本，公开版本显式声明如 `"v1alpha1"`，并通过 `install.Install` 中的 `SetVersionPriority` 控制序列化优先级。
