- 对外稳定版本统一放在 `v1beta1/` 子包内，并在该包中单独声明 `SchemeGroupVersion` 与 `AddToScheme`，顶层只保留内部版本与别名注册。
- 需要特殊处理的字段转换采用“先调用 `autoConvert_*` 再补零长度切片”的模式，以保证与历史手动转换行为一致。
- 默认值填充遵循 `SetDefaults_<Type>(obj *extensionsv1beta1.<Type>)` 命名约定，由 `zz_generated.defaults.go` 在 `addDefaultingFuncs` 中统一注册。
- Scheme 安装通过 `install.Install(scheme)` 集中编排，并在 `init()` 中绑定到 `legacyscheme.Scheme`，避免在业务代码中直接操作 Scheme。
