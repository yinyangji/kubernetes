- 每个版本包通过 `doc.go` 上的 `+k8s:*-gen` 注释声明生成目标，并由同名 `zz_generated.*.go` 文件承载自动生成代码，手写逻辑与生成代码分离。
- 跨包字段标签转换统一以 `AddFieldLabelConversionsForXxx(scheme *runtime.Scheme)` 形式封装并在 `addConversionFuncs` 中集中注册，返回错误时直接向上冒泡。
- 需要特殊处理的字段转换（如 `PodCIDR` 单值↔数组、`StringData→Data` 覆盖、`ReplicationController↔ReplicaSet`）以手写 `Convert_*_To_*` 函数显式实现，而非依赖自动转换。
- 内部类型注册集中在 `register.go` 的 `addKnownTypes` 中通过 `scheme.AddKnownTypes(SchemeGroupVersion, &T{}, ...)` 批量声明，避免分散在各处。
