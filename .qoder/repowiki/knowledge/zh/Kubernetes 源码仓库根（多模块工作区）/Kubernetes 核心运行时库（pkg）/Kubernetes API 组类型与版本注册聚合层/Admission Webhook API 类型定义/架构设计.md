本模块是 Kubernetes 准入控制器（Webhook）协议的数据契约层，按 apimachinery 约定组织：
- `types.go` + `register.go` + `doc.go` 定义内部组 `admission.k8s.io` 的 `AdmissionReview`、`AdmissionRequest`、`AdmissionResponse`、`Operation`、`PatchType` 等核心类型，并通过 `SchemeGroupVersion = runtime.APIVersionInternal` 注册到内部 Scheme。
- `v1/` 与 `v1beta1/` 子包分别对应公开稳定版与 beta 版，每个子包仅持有 `SchemeGroupVersion`、`Resource()` 辅助函数以及指向外部 `k8s.io/api/admission/<version>` 包的 `SchemeBuilder` 引用；默认值注册通过 `zz_generated.defaults.go` 中的 `RegisterDefaults` 注入，由 `init()` 中显式 `localSchemeBuilder.Register(RegisterDefaults)` 触发，从而在生成文件缺失时仍可编译。
- `install/install.go` 作为聚合入口，调用各版本的 `AddToScheme` 并设置版本优先级（v1 > v1beta1），同时向 `legacyscheme.Scheme` 注册，供 apiserver 启动时消费。
- `fuzzer/fuzzer.go` 为 fuzzing 提供随机数据生成器，独立于主类型定义。
- 所有 deepcopy、conversion、defaults、validations 代码均由 `+k8s:deepcopy-gen` / `+k8s:conversion-gen` 等注释驱动生成（`zz_generated.*.go`），手写代码仅保留注解与最小胶水逻辑。
