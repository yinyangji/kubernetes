- 对外版本类型均实现 `runtime.Object` 接口并通过 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解声明，以便被 Scheme 序列化和反序列化。
- 版本到内部类型的转换在各自 `conversion.go` 中手写 `Convert_<ver>_Policy_To_abac_Policy`，先调用 `autoConvert_*` 再补齐语义差异（如 `*` 或空 user/group 映射为 `system:authenticated`）。
- 每个版本目录配套一组 `zz_generated.*` 文件（deepcopy、conversion、model_name），由 apimachinery 代码生成器产出，禁止手工修改。
- 版本包统一提供 `doc.go`（包文档 + openapi 注解）、`register.go`（SchemeBuilder/AddToScheme）、`types.go`（类型定义）、`conversion.go`（转换）四件套结构。
