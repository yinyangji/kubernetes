- 每个 API 版本子包都遵循 `doc.go` + `register.go` + `zz_generated.*.go` 的最小骨架，并在 `register.go` 中暴露同名的 `GroupName`、`SchemeGroupVersion`、`Resource()` 与 `AddToScheme` 符号。
- 当外部版本与内部类型存在结构不兼容（如 v1alpha1 的 `Spec.RuntimeHandler` 映射到内部 `Handler`）时，通过手写 `Convert_*_To_node_*` 函数覆盖自动生成器的默认行为，而非修改内部类型。
- 对外部版本差异（字段名/层级）通过 `field.NormalizationRule` 的正则替换进行归一化，使 validation 层只面对统一的内部结构。
- 所有新增类型均添加 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解，以便自动生成 DeepCopy 实现并纳入 runtime.Object 接口族。
