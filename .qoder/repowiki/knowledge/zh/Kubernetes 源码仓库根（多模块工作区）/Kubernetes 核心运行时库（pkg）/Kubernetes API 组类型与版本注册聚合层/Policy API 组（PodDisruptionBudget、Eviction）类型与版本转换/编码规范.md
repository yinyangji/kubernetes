- 每个版本子包（v1、v1beta1）都独立声明 `GroupName="policy"` 与对应 `SchemeGroupVersion`，并通过 `localSchemeBuilder.Register(RegisterDefaults)` 在 init 中注册默认值，使缺失生成文件时仍可编译。
- 对外部 CRD 类型（位于 `k8s.io/api/policy/vX`）仅通过 `+k8s:conversion-gen-external-types` 注释引用，本模块自身不重复定义这些结构体，所有字段变更以转换函数桥接。
- API 对象的 DeepCopy 与默认值通过 `// +k8s:deepcopy-gen=package` 与 `// +k8s:defaulter-gen=TypeMeta` 注释交由代码生成器处理，手写代码只保留业务逻辑。
- 验证函数统一返回 `field.ErrorList`，并以 `field.NewPath("spec")` / `.Child(...)` 构造路径前缀，复用 `appsvalidation`、`apivalidation`、`unversionedvalidation` 等共享校验器。
