- 每个 API 包遵循标准布局：根目录放 internal 版本类型与 SchemeGroupVersion，`v1alpha1/` 子包通过代理外部 `k8s.io/api/<pkg>/v1alpha1` 的 SchemeBuilder 完成注册。
- 类型通过 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解标记可序列化对象，由 code-generator 产出 `zz_generated.deepcopy.go`。
- 校验函数统一返回 `field.ErrorList`，并以 `field.NewPath(...).Child(...)` 构建带路径前缀的错误定位信息。
- 名称/标识符校验复用 apimachinery 的 DNS 规则：name 用 `IsDNS1123Subdomain`/`IsDNS1035Label`，apiServerID 用 `NameIsDNSSubdomain`。
