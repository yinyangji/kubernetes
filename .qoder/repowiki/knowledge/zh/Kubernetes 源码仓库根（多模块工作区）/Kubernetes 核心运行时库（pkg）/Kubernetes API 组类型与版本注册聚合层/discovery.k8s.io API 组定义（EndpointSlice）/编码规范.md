- 每个版本子包（v1、v1beta1）都独立定义 `GroupName`、`SchemeGroupVersion` 与 `Resource()` 辅助函数，并导出 `AddToScheme` 以被 install 层统一注册。
- 版本包的 `register.go` 中 `init()` 仅注册手写函数（如 `addDefaultingFuncs`），显式注释说明其他注册由 `zz_generated` 文件完成，保证生成文件缺失时仍可编译。
- 顶层 `types.go` 使用 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解使结构体自动实现 `runtime.Object` 接口，并由 `zz_generated.deepcopy.go` 提供 DeepCopy 实现。
- 对外公开的类型通过 `metav1.TypeMeta` + `metav1.ObjectMeta`/`metav1.ListMeta` 组合嵌入，保持与 Kubernetes 元数据约定一致。
- 可选字段统一使用指针类型（如 `*string`、`*bool`、`*int32`）配合 `+optional` 注解，以区分未设置与零值语义。
