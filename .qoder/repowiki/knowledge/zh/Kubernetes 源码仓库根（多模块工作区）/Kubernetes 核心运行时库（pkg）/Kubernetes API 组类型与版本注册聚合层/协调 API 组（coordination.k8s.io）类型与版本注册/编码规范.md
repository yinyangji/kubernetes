- 每个公开版本子包（v1/v1beta1/v1alpha2）只保留 `register.go` 与生成的 `zz_generated.*.go`，类型定义委托给外部 `k8s.io/api/coordination/<version>`，通过引用其 `SchemeBuilder` 完成注册。
- 顶层 `types.go` 中的对象均添加 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解，使生成的 DeepCopy 方法同时实现 Object 接口。
- API 字段通过 `+featureGate=CoordinatedLeaderElection` 注释标记实验性字段，配合 `+optional`/`+required` 等 kubebuilder 标签描述必填性与可见性。
- 校验函数遵循 `ValidateXxx` / `ValidateXxxUpdate` / `ValidateXxxSpec` 三段式命名，先调用 `apimachinery` 的 `ValidateObjectMeta*` 做元数据校验，再追加 spec 级错误到 `field.ErrorList`。
