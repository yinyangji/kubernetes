- 每个版本子包 (`v1`/`v1beta1`) 只保留 `doc.go`、`register.go` 与 code-generator 产出的 `zz_generated.*`，不手写业务逻辑，通过别名方式复用 `k8s.io/api/storagemigration/<ver>` 的 SchemeBuilder。
- 对外暴露的 GroupName 常量在各层保持一致为 `storagemigration.k8s.io`，并通过 `schema.GroupVersion.WithResource/WithKind` 派生 `GroupResource`/`GroupKind`。
- API 对象使用 `+genclient:+genclient:nonNamespaced` 注解声明为非命名空间资源，并在 `Status.Conditions` 上使用 `+patchMergeKey=type` / `+patchStrategy=merge` 以支持条件合并补丁。
- 校验函数统一返回 `field.ErrorList`，通过 `field.NewPath(...)` 构造路径并追加到 `allErrs`，对不可变字段使用 `ValidateImmutableField` 表达变更拒绝。
