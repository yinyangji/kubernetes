模块遵循 Kubernetes API 包标准分层：
- `types.go` 声明内部版本（`runtime.APIVersionInternal`）的 `StorageVersionMigration`/`Spec`/`Status` 结构体，并通过 `+genclient:+genclient:nonNamespaced` 注解生成非命名空间型 REST 资源；`register.go` 暴露 `GroupName="storagemigration.k8s.io"`、`SchemeGroupVersion`、`Kind`/`Resource` 辅助函数以及 `addKnownTypes`。
- `v1/` 与 `v1beta1/` 子包仅做 Scheme 别名转发（引用 `k8s.io/api/storagemigration/v1` 与 `v1beta1`），配合 `zz_generated.*` 文件由 code-generator 产出转换、默认值与验证逻辑，本目录不含手写业务代码。
- `install/install.go` 聚合各版本的 `AddToScheme` 供控制器侧一次性注册。
- `validation/validation.go` 集中实现对象元数据、`spec.resource`（DNS1035/DNS1123）、状态条件互斥与不可变约束等字段级校验，被上层 APIService 在准入阶段调用。
依赖方向单向：`v1`/`v1beta1` → 外部 `k8s.io/api/storagemigration/*` → 本模块内部 `types.go`；`validation` 仅依赖内部类型与 apimachinery 校验工具。
