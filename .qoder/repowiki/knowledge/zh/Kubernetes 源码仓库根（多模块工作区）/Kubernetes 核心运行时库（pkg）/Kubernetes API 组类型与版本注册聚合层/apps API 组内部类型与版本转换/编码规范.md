- 每个公开版本目录都遵循相同文件模板：`doc.go` 声明 codegen 注释、`register.go` 注册 SchemeGroupVersion、`conversion.go` 手写覆盖 auto-conversion、`defaults.go` + `zz_generated.defaults.go`、`zz_generated.validations.go`。
- 对已废弃字段采用 annotation ↔ field 双向转换模式：在 conversion 中读写 `appsv1.Deprecated*` 注解以维持向后兼容，并在注释中标注 TODO 以便删除旧版本后移除。
- 状态字段校验统一返回 `field.ErrorList`，通过 `apivalidation.ValidateNonnegativeField` 检查非负性，并用 `WithOrigin("immutable")` 标记不可变字段变更错误。
- 选择器与 PodTemplateSpec 校验先通过 `metav1.LabelSelectorAsSelector` 构建 selector，再断言其匹配 template labels，且强制 `RestartPolicy=Always`、禁止 `ActiveDeadlineSeconds`。
