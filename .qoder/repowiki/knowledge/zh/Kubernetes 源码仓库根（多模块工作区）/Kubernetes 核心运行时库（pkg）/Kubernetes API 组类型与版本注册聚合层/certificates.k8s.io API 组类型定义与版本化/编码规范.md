- 每个版本子包都遵循相同的文件骨架：doc.go（包注释）、register.go（SchemeGroupVersion + Kind/Resource 工厂）、conversion.go、defaults.go、zz_generated.conversion.go、zz_generated.defaults.go、zz_generated.validations.go，便于 code-generator 增量更新。
- 对外暴露的 API 类型统一实现 runtime.Object 接口（通过 `+k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注解），并配套提供同名的 *List 类型。
- 版本优先级通过 install.Install 中的 `scheme.SetVersionPriority` 显式声明，而非依赖导入顺序，确保 v1 始终为首选版本。
- 校验逻辑集中在 validation 包，以 `ValidateXxxCreate/Update/StatusUpdate` 三件套形式暴露，并通过 options 结构体控制兼容性开关，避免在各版本间复制校验分支。
