模块按 Kubernetes API 分层约定组织：
- `pkg/apis/autoscaling` 根包声明内部 GroupName=`autoscaling`，并通过 `register.go` 的 `addKnownTypes` 将 `Scale`、`HorizontalPodAutoscaler`、`HorizontalPodAutoscalerList` 注册到内部 Scheme；`types.go` 是内部对象模型的唯一来源，`helpers.go` 提供跨版本 round-trip 注解清理工具。
- `v1/`、`v2/` 子包分别对应 `k8s.io/api/autoscaling/v1` 与 `/v2` 外部版本，每个子包包含 `register.go`（绑定外部 `SchemeBuilder`）、`conversion.go`（手写双向转换）、`defaults.go` + `zz_generated.defaults.go`、`zz_generated.validations.go`、`zz_generated.conversion.go`（由 deepcopy/conversion generator 生成），以及对应的 `_test.go`。
- `validation/` 子包集中实现 `ValidateScale`、`ValidateHorizontalPodAutoscaler*` 等手写校验器，并复用 `rest.DeclarativeValidation` 做声明式校验组合。
- `fuzzer/` 为 fuzzing 测试提供随机构造器，`install/` 暴露安装入口。
依赖方向单向：`v1`/`v2` → 内部 `autoscaling` → `apimachinery`/`core`；内部包不反向依赖任何外部版本包。
