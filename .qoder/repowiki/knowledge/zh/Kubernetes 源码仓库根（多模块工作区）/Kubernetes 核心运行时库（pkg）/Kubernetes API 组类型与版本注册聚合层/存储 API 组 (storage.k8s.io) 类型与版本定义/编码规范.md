- 每个已发布版本子包都遵循固定文件布局：`doc.go`（group 注解）、`register.go`（定义 `SchemeGroupVersion` 与 `AddToScheme`）、`defaults.go`（手写默认值）、`zz_generated.*`（自动生成），新增版本只需复制该模板。
- 对外暴露的 GroupName 常量与 `SchemeGroupVersion` 在每个版本包内重复声明，但 `vN/register.go` 通过引用 `k8s.io/api/storage/vN` 的 `SchemeBuilder` 复用其注册逻辑，避免重复维护。
- 所有资源对象均附带 `// +k8s:deepcopy-gen:interfaces=k8s.io/apimachinery/pkg/runtime.Object` 注释，并配套提供 `<Kind>List` 列表类型，统一实现 `runtime.Object` 接口。
- 验证函数成对出现：`ValidateXxx` 负责创建路径校验，`ValidateXxxUpdate` 在此基础上追加 `ValidateImmutableField(...).WithOrigin("immutable").MarkCoveredByDeclarative()` 标记不可变字段。
- 可选布尔字段使用 `*bool` 指针并在注释中说明 nil 语义（如 `AllowVolumeExpansion`、`PodInfoOnMount`、`SELinuxMount`），配合 `+optional` 标签交由生成器处理 JSON 序列化。
- feature-gated 字段通过 `// +featureGate=<Gate>` 注释标注，并在 `validation` 层用 `utilfeature.DefaultFeatureGate.Enabled(features.XXX)` 控制必填或取值范围。
