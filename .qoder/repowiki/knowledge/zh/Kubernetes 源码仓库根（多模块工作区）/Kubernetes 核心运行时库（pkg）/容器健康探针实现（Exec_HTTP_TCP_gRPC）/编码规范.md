- 每个协议子包都定义一个 `Prober` 接口并提供 `New()` 构造函数，内部结构体名采用 `<protocol>Prober` 形式。
- 对外同时暴露 `Do*Probe` 纯函数（如 `DoHTTPProbe`、`DoTCPProbe`），使调用方可绕过接口直接复用探测逻辑。
- 网络 I/O 全部通过 `probe.ProbeDialer()` 获取的 Dialer 发起，避免在各子包重复配置 SO_LINGER。
- 所有底层错误在探针层统一转换为 `probe.Failure` 并附带错误文本作为 message，仅成功路径返回空 error。
- 响应体/输出读取限制在 10KB（常量 `maxRespBodyLength` / `maxReadLength`），超限记录 V(4) 日志但不视为失败。
