本目录是 KCM 中所有存储相关控制器的扁平集合，每个子包实现一个独立的 Controller（PersistentVolumeController、AttachDetachController、EphemeralVolumeController、PVC/PV/VAC ProtectionController、SELinuxWarningController），它们不互相调用，而是通过监听 API Server 的同一组资源（Pod、PV、PVC、Node、CSIDriver 等）并写入其 status/finalizer 来间接协作：persistentvolume 负责 PV↔PVC 绑定与动态供给；attachdetach 消费已绑定的 VolumeAttachment 完成跨节点挂载；ephemeral 为 inline EphemeralVolumeSource 自动生成 PVC；protection_controllers 通过 finalizer 阻止未就绪资源的删除；selinux_warning 仅做只读检测与事件上报。各控制器共享 pkg/controller/volume/common 中的通用工具函数，并通过 metrics/events 子包暴露指标与事件，最终由 KCM 启动时按 feature gate 选择性注册。
