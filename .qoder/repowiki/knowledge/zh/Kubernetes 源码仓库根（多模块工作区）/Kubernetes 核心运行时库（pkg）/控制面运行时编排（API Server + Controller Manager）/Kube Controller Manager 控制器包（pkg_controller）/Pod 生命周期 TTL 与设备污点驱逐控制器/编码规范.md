- 控制器构造器统一命名为 New*，并在构造函数中完成 queue/informer/recorder 初始化，Run(ctx, workers) 负责启动 worker goroutine 与 cache.WaitForNamedCacheSyncWithContext 等待同步。
- 事件处理器 AddFunc/UpdateFunc/DeleteFunc 只做轻量判断并将对象 key 通过 controller.KeyFunc 放入 rate-limiting queue，所有阻塞式 API 调用集中在 processNextWorkItem/worker 路径。
- 错误处理采用 handleErr 模式：nil 则 Forget(key)，否则 utilruntime.HandleError 后再 AddRateLimited(key) 重试。
- 对外暴露的 metrics 在 New/Run 入口处调用各自 metrics.Register() 进行一次性注册，避免重复注册。
- 需要延迟执行的任务通过 queue.AddAfter(key, duration) 而非 sleep 实现，确保可被取消和调度。
- 涉及并发写共享 map 的路径使用 sync.Mutex 保护，读取路径在加锁前做快照或先检查再释放锁以减少持锁时间。
