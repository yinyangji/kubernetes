- 所有对外子组件均定义最小 interface（如 `Reconciler`、`DesiredStateOfWorldPopulator`、`NodeStatusUpdater`），由 New 函数返回具体 struct 实现，便于测试注入 fake。
- 日志统一使用 `klog.FromContext(ctx)` 获取 logger，并在关键路径上附带 `klog.KRef` / `klog.KObj` / `klog.KObjSlice` 结构化字段。
- 错误处理遵循“忽略 exponentialbackoff.IsExponentialBackoff 错误，记录其他错误”的模式，避免对预期重试产生噪音日志。
- Informer 事件回调一律做 nil/type 断言后返回，未调度 Pod（`Spec.NodeName == ""`）直接跳过，不进入 volume 处理流程。
- DSW/ASW 的状态变更通过显式方法（`MarkVolumeAsAttached`、`DeletePod`、`SetVolumesMountedByNode` 等）进行，禁止直接读写底层 map。
