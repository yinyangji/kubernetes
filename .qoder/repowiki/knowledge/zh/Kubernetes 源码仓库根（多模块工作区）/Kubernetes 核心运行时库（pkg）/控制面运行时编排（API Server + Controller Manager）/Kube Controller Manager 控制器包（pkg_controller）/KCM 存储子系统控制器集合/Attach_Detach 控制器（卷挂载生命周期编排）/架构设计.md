模块以 `attach_detach_controller.go` 为入口，通过 `NewAttachDetachController` 组装四个并行 goroutine：
- `populator.DesiredStateOfWorldPopulator`：周期性扫描 Pod/PV/PVC informer，将可附加卷写入 `cache.DesiredStateOfWorld`。
- `reconciler.Reconciler`：周期比对 DSW 与 `cache.ActualStateOfWorld`，先触发 Detach、再触发 Attach，调用 `operationexecutor.OperationExecutor` 异步执行，并通过 `statusupdater.NodeStatusUpdater` 写回 Node.Status。
- PVC workqueue worker：监听 PVC 绑定事件，按 PodPVCIndexer 反查 Pod 并更新 DSW。
- Informer event handlers：Pod/Node/PVC 变更直接调用 `util.ProcessPodVolumes` 增量更新 DSW。

内部分层清晰：`cache/` 提供进程内内存缓存（DSW/ASW），`reconciler/` 实现差异计算与动作派发，`populator/` 负责从 informer 填充 DSW，`statusupdater/` 封装 Node 状态写入，`metrics/` 暴露 Prometheus 指标，`config/v1alpha1/` 承载配置类型及 conversion/deepcopy 生成代码，`util/` 提供 volume spec 构造、action 判定等共享逻辑。依赖方向单向：controller → populator/reconciler/statusupdater/metrics/util → cache；对外仅暴露 `AttachDetachController` 接口供上层启动。
