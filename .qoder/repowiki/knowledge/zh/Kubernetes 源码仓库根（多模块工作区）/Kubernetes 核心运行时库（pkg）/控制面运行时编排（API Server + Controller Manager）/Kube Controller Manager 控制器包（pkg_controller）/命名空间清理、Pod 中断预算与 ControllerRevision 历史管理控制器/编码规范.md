- 控制器构造器成对出现：对外 `NewXxxController` 使用真实时钟/默认超时，内部 `NewXxxControllerInternal` 暴露 clock.WithTicker 与可配置超时供测试注入。
- 工作队列统一使用 `workqueue.NewTypedRateLimitingQueueWithConfig` 配合 `DefaultTypedControllerRateLimiter` 或自定义 MaxOfRateLimiter，并在 Run 中用 `cache.WaitForNamedCacheSyncWithContext` 等待所有 informer 同步后再启动 worker。
- 事件处理器中对 DeleteFunc 的 tombstone 使用 `cache.DeletedFinalStateUnknown` 类型断言进行安全恢复，避免丢失已删除对象的 key。
- 错误处理统一走 `utilruntime.HandleErrorWithContext`，并将带估计剩余时间的 `*deletion.ResourcesRemainingError` 作为特殊分支做指数退避重入队。
- 日志统一通过 `klog.FromContext(ctx)` 获取 logger，并以结构化字段记录 namespace、key、duration 等上下文信息。
