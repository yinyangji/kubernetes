- 每个控制器子模块在自身 `config/v1alpha1` 下提供独立的 types.go/register.go/defaults.go，并在顶层 `apis/config/v1alpha1/defaults.go` 中注册默认值以参与全局 Scheme。
- 控制器普遍采用 Informer + WorkQueue 驱动，并通过 `controller.NewPodControllerRefManager` / `NewReplicaSetControllerRefManager` 等工厂复用 ownerRef 认领逻辑。
- 可选行为通过 `feature.DefaultFeatureGate` 开关，新增控制器需先在 features 注册 FeatureGate 再在 Run 函数中判断是否启用。
- 指标统一放在子模块 `metrics/metrics.go` 中定义 prometheus 计数器/直方图，由外部入口汇总暴露。
