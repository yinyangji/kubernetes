- 每个控制器包都包含 `config/types.go` 与 `config/v1alpha1/{conversion,defaults,register,doc}.go`，以 v1alpha1 版本暴露可配置的控制器参数并提供默认值与转换逻辑。
- 控制器入口统一采用 `New*` 构造器接收 informer 与 clientset，内部创建 `record.NewBroadcaster` 与 `TypedRateLimitingInterface`，并在 `Run(ctx, workers)` 中等待缓存同步后启动 worker goroutine。
- 通过 `cache.InformerSynced` 字段（如 `jobStoreSynced`、`podStoreSynced`）暴露给测试注入，配合 `cache.WaitForNamedCacheSyncWithContext` 确保 informer 就绪后再开始处理。
- 使用 `schema.GroupResource` 常量标识被监控的资源，并在 feature gate 开启时将对应 store 注册到 `consistencyutil.ConsistencyStore` 以追踪写入时间戳。
- 单元测试通过替换 `syncHandler`、`enqueue*`、`clock`、`now` 等字段来注入行为，避免真实 API 调用。
