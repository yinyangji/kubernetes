- 对同一资源的多个 API 版本通过泛型 `T clusterTrustBundle` 加 `clusterTrustBundleHandlers` 接口解耦，各版本仅实现 create/update/contains/getName 四个方法。
- 对外暴露统一 `PublisherRunner` 接口（Run(context.Context)），内部构造器返回该接口，隐藏版本差异。
- informer 初始化时使用 `fields.OneTermEqualSelector("spec.signerName", signerName)` 做字段过滤，避免拉取全量资源。
- 工作队列采用 `workqueue.NewTypedRateLimitingQueueWithConfig` 配合 `DefaultTypedControllerRateLimiter`，错误路径走 `AddRateLimited`，成功路径走 `Forget`。
- 指标记录将 error 转换为 HTTP status code 字符串作为 label，默认 "500"，无错时记 "200"。
