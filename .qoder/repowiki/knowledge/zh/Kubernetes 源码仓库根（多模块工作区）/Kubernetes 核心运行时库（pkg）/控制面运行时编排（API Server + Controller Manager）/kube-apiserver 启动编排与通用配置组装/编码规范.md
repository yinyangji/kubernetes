- 每个可插拔子系统的选项类型统一实现 `AddFlags(fs)` + `Validate() []error` + `To*Config()` + `ApplyTo(...)` 四件套，由上层在固定顺序中依次调用完成装配。
- 可选功能通过 `utilfeature.DefaultFeatureGate.Enabled(apiserverfeatures.Xxx)` 判断后再注册 PostStartHook 或控制器，避免硬编码开关。
- 运行时后台协程一律通过 `GenericAPIServer.AddPostStartHookOrDie` / `AddPreShutdownHookOrDie` 注册，确保生命周期与 HTTP handler 就绪状态对齐。
- 认证/授权配置文件支持热重载：先 `loadAuthenticationConfig` 解码，再用 `apiservervalidation.ValidateAuthenticationConfiguration` 校验，最后调用返回的 `updateAuthenticationConfig` 闭包原子替换。
