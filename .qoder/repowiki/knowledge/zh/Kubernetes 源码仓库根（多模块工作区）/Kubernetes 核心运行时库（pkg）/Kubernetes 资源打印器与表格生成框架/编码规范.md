- 每个资源类型成对注册：同一份 `*ColumnDefinitions` 同时传给 `h.TableHandler(..., printXxx)` 和 `h.TableHandler(..., printXxxList)`，保证单例与列表共用列定义。
- 列定义优先从对应 API 类型的 SwaggerDoc 抽取 Description，避免手写重复描述。
- 可选扩展列统一设置 `Priority: 1`，以便 `Wide=false` 时自动隐藏，核心列保持 `Priority == 0`。
- 长列表字段使用 `listWithMoreString` 或类似模式截断并在尾部追加 `+ N more...`，防止终端溢出。
- 时间戳格式化统一走 `translateTimestampSince`/`translateMicroTimestampSince` 返回 `duration.HumanDuration`，空值回退 `<unknown>`。
