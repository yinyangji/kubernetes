- OpenAPI 文件命名采用 `<resource-path>_openapi.json` 形式，并按 group/version 拆分为独立文件，同时提供聚合根文件（如 `apis_openapi.json`、`swagger.json`）。
- Discovery JSON 文件名严格映射 REST 路径：`apis.json`、`api.json`、`apis__<group>.json`、`apis__<group>__<version>.json`，便于直接对应 `/apis` 树结构。
- API 规则例外清单使用 `API rule violation: <RULE>,<PACKAGE>,<TYPE>,<FIELD>` 单行格式，并在每个维度上按字母序排序，禁止隐藏于生成变更提交中。
