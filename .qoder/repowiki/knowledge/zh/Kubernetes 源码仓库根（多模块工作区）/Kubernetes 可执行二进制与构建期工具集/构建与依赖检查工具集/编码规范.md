- 每个子目录是一个独立的 `package main`，入口函数为 `main()`，通过 `os.Args` 或 `flag` 解析参数而非 cobra 子命令。
- 外部命令调用统一封装为 `runCommand` / `runCommandInDir`，将 `stdout+stderr` 合并返回并在失败时附带原始输出以便诊断。
- JSON 配置采用 spec/status 双段结构（如 `Unwanted.Spec` 与 `Unwanted.Status`），工具只写入 status 字段并与磁盘上的 status 做 diff，以此驱动 PR 审查。
- 所有不可恢复错误均通过 `log.Fatalf` / `log.Fatal` 终止进程并以非零退出码表达失败，便于 CI 集成。
