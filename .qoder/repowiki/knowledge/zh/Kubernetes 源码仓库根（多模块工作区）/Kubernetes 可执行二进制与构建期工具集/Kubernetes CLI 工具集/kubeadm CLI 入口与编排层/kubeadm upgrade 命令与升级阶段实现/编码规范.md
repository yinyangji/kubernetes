- CLI 标志统一通过 cmdutil.ValueFromFlagsOrConfig 从命令行与 UpgradeConfiguration 合并解析，并对类型进行显式断言，失败时返回 cmdutil.TypeMismatchErr。
- 错误信息以 `[upgrade/<phase>] FATAL` 前缀包裹 errors.Wrapf，便于用户快速定位失败阶段。
- 每个 cobra 子命令采用 `newCmdXxx(flags)` 工厂函数返回 *cobra.Command，并在 RunE 中构造数据对象后交给 workflow.Runner 执行。
- 测试通过包级 TestMain 调用 pkiutiltesting.RunWithPrivateKeyFixtureDirectory 集中初始化 PKI fixture 目录，避免在每个测试中重复搭建环境。
