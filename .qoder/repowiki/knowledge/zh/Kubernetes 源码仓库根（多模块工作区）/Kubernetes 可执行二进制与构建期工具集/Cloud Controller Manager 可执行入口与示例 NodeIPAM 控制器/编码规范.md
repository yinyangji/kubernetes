- 外部控制器的标准接入模式：定义一个 struct 持有 Options+Configuration，暴露 `StartXxxControllerWrapper(initContext, completedConfig, cloud) app.InitFunc` 方法，返回的 InitFunc 内完成参数校验并构造实际 controller.Interface。
- 云厂商插件通过空白导入 `_ "k8s.io/legacy-cloud-providers/<provider>"` 注册到全局 `cloudprovider.Interface` 工厂，而非显式 new。
- 对集群网络参数的解析统一走 `netutils.ParseCIDRs` / `IsDualStackCIDRs`，并在多 CIDR 场景下强制要求 IPFamily 不同，否则返回错误。
