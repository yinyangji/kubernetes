本目录是 kube-apiserver 的单一入口：apiserver.go 通过 import 副作用加载 component-base 的 JSON 日志格式、prometheus client-go 指标与版本指标，随后调用 app.NewAPIServerCommand() 把 options（app/options）与 server 引导（app/server.go → aggregator/config）组装成完整的 cobra 命令树；app.NewAPIServerCommand 返回的命令由 cli.Run 统一执行生命周期（ParseFlags → Run），从而将“命令行选项定义/校验”和“服务器链装配”两个子模块串联起来，对外只暴露一个 kube-apiserver 可执行文件。
