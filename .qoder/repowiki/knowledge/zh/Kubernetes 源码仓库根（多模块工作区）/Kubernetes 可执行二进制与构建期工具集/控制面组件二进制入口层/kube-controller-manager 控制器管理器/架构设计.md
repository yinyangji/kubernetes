本模块是 KCM 的顶层装配层，将三个子域按职责分层串联：
- `controller_bootstrap_and_options` 提供 CLI 入口 (`cmd/kube-controller-manager/controller-manager.go`)、全局选项解析以及 Bootstrap/TokenCleaner 等内嵌控制器的构造；
- `controller_framework_and_utils` 定义统一的 Controller 接口与描述符注册机制，各业务控制器通过 `app/options/*.go` 暴露选项并实现该接口；
- `admission_and_plugins` 负责 EndpointSlice、EndpointSliceMirroring、ValidatingAdmissionPolicyStatus 等控制器的描述符注册，并将持久卷插件汇总供 attach/detach 等控制器消费。
三者通过 `app/controllermanager.go` 中的控制器发现与启停循环在运行时组装，形成“选项 → 描述符 → 实例化 → 运行”的统一流水线。
