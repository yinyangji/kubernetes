- 每个控制器通过 newXxxControllerDescriptor() 返回 *ControllerDescriptor 并传入 register() 完成注册，register 内部对空指针、空名、重复名、重复 alias、缺失 constructor 做 panic 级校验。
- 需要 feature gate 控制的控制器将其 Feature 放入 ControllerDescriptor.requiredFeatureGates，由 BuildController 统一拦截并返回 nil 表示禁用，而非在构造器内自行判断。
- runFuncSlice 模式用于把多个 runFunc 并行启动并用 sync.WaitGroup 等待全部结束，确保 Run 返回时所有子协程已终止。
- API Group 通过空白导入 pkg/apis/<group>/install 触发全局 scheme 注册，避免显式调用注册函数。
